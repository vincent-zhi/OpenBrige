# Custom Agent Profile Example

This example shows how to create a custom agent profile for OpenBrige.

## Usage

1. Copy this directory to `.openbrige/profiles/my-custom-agent/`
2. Edit `profile.yaml` to match your agent's CLI
3. Restart OpenBrige: `openbrige start`

## Files

- `plugin.json` - Plugin manifest
- `profile.yaml` - Agent profile configuration
